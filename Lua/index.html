Lua

Tipos de Variáveis

number → números inteiros ou decimais
string → textos
boolean → verdadeiro ou falso
table → listas ou dicionários
nil → ausência de valor
function → funções

Operadores e Métodos

.. → concatenação de strings
# → comprimento de uma tabela ou string
table.insert(tabela, valor) → adiciona valor a uma tabela
table.remove(tabela, indice) → remove elemento
string.upper(texto) → converte para maiúsculas
string.lower(texto) → converte para minúsculas
tonumber(valor) → converte para número
tostring(valor) → converte para string
print(valor) → imprime na tela
pairs(tabela) → iteração sobre tabelas
ipairs(tabela) → iteração sequencial
math.floor(valor) → arredonda para baixo
math.ceil(valor) → arredonda para cima
math.random(min, max) → retorna número aleatório
os.time() → retorna tempo atual
