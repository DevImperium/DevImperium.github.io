Java

Tipos de Variáveis

int / double → números inteiros ou decimais
String → textos
boolean → verdadeiro ou falso
Array / List → listas de valores
null → ausência de valor
Métodos → funções dentro de classes

Operadores e Métodos

+ → concatenação de strings
.length() → comprimento de strings / arrays
lista.add(valor) → adiciona valor a uma lista
lista.remove(indice) → remove elemento
texto.toUpperCase() → converte para maiúsculas
texto.toLowerCase() → converte para minúsculas
Integer.parseInt(valor) → converte para número
String.valueOf(valor) → converte para string
System.out.println(valor) → imprime na tela
for (var v : lista) → iteração sobre listas
for (int i = 0...) → iteração sequencial
Math.floor(valor) → arredonda para baixo
Math.ceil(valor) → arredonda para cima
Math.random() → retorna número aleatório
System.currentTimeMillis() → retorna tempo atual
