Java 







Tipos de Dados:


int → números inteiros (ex: 10, -5)

float → números decimais simples (ex: 3.14f)

double → números decimais com maior precisão

char → caractere único (ex: 'a')

String → texto (objeto imutável)

boolean → verdadeiro (true) ou falso (false)

void → ausência de retorno (usado em métodos)

var → tipo inferido automaticamente (Java 10+)

long / short / byte → variações numéricas

final → define constante (valor imutável)



Operadores:


Aritméticos: → +, -, *, /, %, ++, --

Relacionais: → ==, !=, <, >, <=, >=

Lógicos: → && (E), || (OU), ! (NÃO)

Atribuição: → =, +=, -=, *=, /=, %=

Bitwise: → &, |, ^, ~, <<, >>

Outros: → ?: (ternário), instanceof, new



Entrada e Saída:


System.out.println() → imprime com quebra de linha

System.out.print() → imprime sem quebrar linha

Scanner → lê entrada do usuário (ex: Scanner sc = new Scanner(System.in);)

nextLine() → lê linha completa (String)

nextInt(), nextDouble() → lê números



Estruturas de Controle:


if / else → decisão condicional

switch / case → seleção de casos (suporta Strings no Java 7+)

for → laço com contador

while → laço com condição

do / while → laço com verificação no final

break / continue → controla fluxo dos laços



Métodos (Funções):


void nome() → método sem retorno

tipo nome(parâmetros) → método com retorno

return → retorna valor

static → pertence à classe, não à instância

this → referência ao próprio objeto

recursivo → método que chama a si mesmo

sobrecarga → vários métodos com o mesmo nome, parâmetros diferentes



Classes e Objetos:


class → define uma classe

public / private / protected → níveis de acesso

constructor → método com o mesmo nome da classe

new → cria novo objeto

this → referência ao próprio objeto

extends → herança de outra classe

super → chama construtor ou método da classe-pai

abstract / interface → classes e contratos de métodos



Strings (java.lang.String):


.length() → tamanho da string

.isEmpty() → verifica se está vazia

.concat(str) → concatena texto

.substring(i, j) → extrai parte da string

.replace(a, b) → substitui texto

.toUpperCase() / .toLowerCase() → maiúsculas/minúsculas

.equals(str) → compara conteúdo

.charAt(i) → retorna caractere na posição

.contains(str) → verifica se contém o texto



Arrays e Coleções:


int[] nums = new int[5]; → cria vetor de 5 inteiros

ArrayList<T> → lista dinâmica (import java.util.*)

.add(x) → adiciona elemento

.remove(i) → remove elemento

.get(i) → acessa elemento

.size() → número de elementos

HashMap<K,V> → armazena pares chave/valor

.put(k, v) → adiciona valor

.get(k) → obtém valor pela chave

.containsKey(k) → verifica existência da chave



Tratamento de Exceções:


try / catch → captura erros em tempo de execução

finally → executa sempre, mesmo com erro

throw → lança exceção manualmente

throws → declara que o método pode lançar exceções
