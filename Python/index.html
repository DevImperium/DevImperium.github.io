      Variáveis:
      
      
      
  int → números inteiros
  
  
  float → números decimais
  
  
  str → textos (strings)
  
  
  bool → valores booleanos (verdadeiro ou falso)
  
  
  list → lista que pode mudar
  
  
  tuple → lista que não pode mudar
  
  
  dict → pares de chave e valor
  
  
  set → conjunto sem repetições
  
  
  range → sequência de números
  
  
  complex → números complexos (com j)
  
  
  bytes → dados binários imutáveis
  
  
  bytearray → dados binários mutáveis
  
  
  NoneType → ausência de valor
  
  
  object → base de todos os objetos
  
  
  type → mostra o tipo de um objeto
      
      
      
      Métodos:
      
      
      
  .upper() → Converte os caracteres pra maiúsculas
  
  
  .lower() → converte todos os caracteres pra minúsculas
  
  
  .capitalize() → a primeira letra fica maiúscula e o resto minúscula
  
  
  .title() → coloca a primeira letra de cada palavra em maiúscula
  
  
  .strip() → remove espaços ou caracteres extras no inicio e no fim
  
  
  .replace(a, b) → Troca todos os "a" por "b"
  
  
  casefold() → converte tudo para minúsculo (mais forte que lower)
  
  
  center(width, fillchar) → centraliza o texto com tamanho fixo width e preenche com fillchar
  
  
  count(sub[, start[, end]]) → conta quantas vezes aparece
  
  
  encode(encoding="utf-8", errors="strict") → retorna a string codificada
  
  
  endswith(suffix[, start[, end]]) → retorna True se a string terminar com suffix
  
  
  expandtabs(tabsize=8) → subistitui tabulações por espaços
  
  
  find(sub[, start[, end]]) → retorna o índice da primeira occorência de sub, ou -1 se não achar
  
  
  format(*args, **kwargs) → formata a string usando chaves {}
  
  
  format-map(mapping) → igual ao format() mas usando um dicionario
  
  index(sub[, start[, end]]) → igual ao find() mas gera erro se não achar
  
  
  isalnum() → retorna true se todos os caracteres forem letras ou números
  
  
  isalpha() → retorna true se todos os caracteres forem letras
  
  
  isascii → retorna true se todos os forem caracteres ASCII (A-Z, 0-9, etc)
  
  
  isdecimal() → retorna true se todos os caracteres forem numeros decimais
  
  
  isdigit() → retorna true se todos os caracteres forem digitos (inclue expoentes, subscritos, etc)
  
  
  isidentifier() → retorna true se uma string for o nome de uma variável válida em Python
  
  
  islower() → retorna true se todos os caracteres forem minúsculos
  
  
  isnumeric() → retorna true se todos os caracteres forem todos numéricos
  
  
  isprintable() → retorna true se todos os caracteres forem imprimíveis (print)
  
  
  isspace() → retorna true se todos forem espaços ou caracteres de espaço em branco
  
  
  istitle() → retorna true se cada palavra começar com letra maiúscula
  
  
  isupper() → retorna true se todos os caracteres forem maiúsculos
  
  
  join(iterable) → junta os elementos de um iterável em uma string usando o texto atual como separador
  
  
  ljus(width, fillchar=' ') → alinha o texto a esquerda e preenche com fillchar
  
  
  lower() → converte todos os caracteres para minúsculo
  
  
  lstrip([chars]) → remove espaços ou caracteres dados (do inicio)
  
  
  maketrans(x, y, z) → cria uma tabela de tradução para translate
  
  
  partition(sep) → divide em 3 partes: antes, separador e depois (primeira occorência)
  
  
  removeprefix(prefix) → remove um prefixo se existir
  
  
  removesuffix(suffix) → remove um sufixo se existir
  
  
  replace(old, new[, count]) → troca old por new (count se quiser limitar)
  
  
  rfind(sub[, start[, find]]) → igual find() mas procura da direita pra esqurda
  
  
  rindex(sub[, start[, end]]) → igual index mas pela direita
  
  
  rjust(width, fillchar=' ' → alinha à direita
  
  
  rpartition(sep) → divide em 3 partes a partir da ultima occorência de sep
  
  
  rsplit(sep=None, maxsplit=-1 → divide da esquerda para a direita
  
  
  rsplit([chars]) → remove espaços ou caracteres dados do fim
  
  
  split(sep=None, maxsplit=-1) → divide uma lista usando o separador (da esquerda para a direita)
  
  
  splitlines(keepends=False) → divide uma string em linhas \n, \r, etc
  
  
  startswith(prefix[, start[, end]]) → retorna true se a string começar com prefix
  
  
  strip([chars]) → remove espaços ou caracteres dados no inicio e fim
  
  
  swapcase() → inverte maiúsculas/minúsculas
  
  
  translate(table) → subistitui caracteres com base em uma tabela (maketrans)
  
  
  zfill(width) → preenche uma string com zeros a esquerda até atingir o tamanho indicado
  
  
  Listas (list):


append(x) → adiciona um item ao final da lista


extend(iterable) → adiciona vários itens (de um iterável) ao final da lista


insert(i, x) → insere o item x na posição i


remove(x) → remove a primeira ocorrência do valor x


pop([i]) → remove e retorna o item na posição i (ou o último se não for passado)


clear() → remove todos os itens da lista


index(x[, start[, end]]) → retorna o índice do primeiro item igual a x


count(x) → conta quantas vezes o valor x aparece na lista


sort(key=None, reverse=False) → ordena a lista


reverse() → inverte a ordem dos elementos da lista


copy() → cria uma cópia rasa da lista



Tuplas (tuple):


count(x) → conta quantas vezes o valor x aparece


index(x[, start[, end]]) → retorna o índice do primeiro valor igual a x



Dicionários (dict):


get(key[, default]) → retorna o valor da chave, ou default se não existir


items() → retorna uma lista de tuplas (chave, valor)


keys() → retorna todas as chaves


values() → retorna todos os valores


pop(key[, default]) → remove e retorna o valor da chave


popitem() → remove e retorna o último par (chave, valor)


setdefault(key[, default]) → retorna o valor da chave, cria se não existir


update([other]) → atualiza o dicionário com pares de outro dicionário


clear() → remove todos os itens do dicionário


copy() → retorna uma cópia rasa do dicionário



Conjuntos (set):


add(elem) → adiciona um elemento ao conjunto


clear() → remove todos os elementos


copy() → retorna uma cópia rasa do conjunto


difference(*others) → retorna os elementos que não estão nos outros conjuntos


difference_update(*others) → remove os elementos presentes em outros conjuntos


discard(elem) → remove o elemento, se existir


intersection(*others) → retorna a interseção com outros conjuntos


intersection_update(*others) → mantém apenas elementos comuns


isdisjoint(other) → retorna True se não houver elementos em comum


issubset(other) → True se for subconjunto de outro


issuperset(other) → True se for superconjunto de outro


pop() → remove e retorna um elemento aleatório


remove(elem) → remove o elemento (gera erro se não existir)


symmetric_difference(other) → retorna elementos que estão em apenas um dos conjuntos


union(*others) → retorna a união dos conjuntos


update(*others) → adiciona elementos de outros Conjuntos
