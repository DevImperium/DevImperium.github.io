Python

Tipos de Variáveis

int / float → números inteiros ou decimais
str → textos
bool → verdadeiro ou falso
list / dict → listas ou dicionários
None → ausência de valor
def → funções

Operadores e Métodos

+ → concatenação de strings
len() → comprimento de uma lista ou string
lista.append(valor) → adiciona valor a uma lista
lista.pop(indice) → remove elemento
texto.upper() → converte para maiúsculas
texto.lower() → converte para minúsculas
int(valor) → converte para número
str(valor) → converte para string
print(valor) → imprime na tela
for chave, valor in dict.items() → iteração sobre dicionários
for i, v in enumerate(lista) → iteração sequencial
math.floor(valor) → arredonda para baixo
math.ceil(valor) → arredonda para cima
random.randint(min, max) → retorna número aleatório
time.time() → retorna tempo atual
