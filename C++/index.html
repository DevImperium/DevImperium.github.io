C++

Tipos de Dados

int → números inteiros (ex: 10, -5)
float → números decimais simples (ex: 3.14f)
double → números decimais com maior precisão
char → caractere único (ex: 'a')
string → texto (necessário incluir <string>)
bool → verdadeiro (true) ou falso (false)
void → ausência de valor (usado em funções)
auto → tipo deduzido automaticamente pelo compilador
long / short → variações do tipo int
unsigned → números sem sinal (apenas positivos)

Operadores

Aritméticos → +, -, *, /, %, ++, --
Relacionais → ==, !=, <, >, <=, >=
Lógicos → && (E), || (OU), ! (NÃO)
Atribuição → =, +=, -=, *=, /=, %=
Bitwise → &, |, ^, ~, <<, >>
Outros → ?: (ternário), sizeof, typeid

Entrada e Saída

cout → imprime texto
cin → lê entrada do usuário
endl → quebra de linha
getline(cin, var) → lê linha inteira incluindo espaços

Estruturas de Controle

if / else → decisão condicional
switch / case → seleção de casos
for → laço de repetição com contador
while → laço com condição no início
do / while → laço com condição no final
break / continue → controla fluxo do laço

Funções

void nome() → função sem retorno
tipo nome(parâmetros) → função com retorno
return → retorna valor
passagem por referência (&) → altera valor original
inline → otimiza chamadas pequenas
recursiva → chama a si mesma
default parameters → valores padrão nos argumentos

Classes e Objetos

class → define uma classe
public / private / protected → níveis de acesso
constructor → função com o mesmo nome da classe
destructor (~Classe) → executado ao destruir o objeto
this → ponteiro para o próprio objeto
new / delete → aloca e libera memória
virtual → permite polimorfismo
herança → classe pode herdar outra

Strings (std::string)

.length() → retorna tamanho da string
.empty() → verifica se está vazia
.append(str) → adiciona texto no fim
.insert(pos, str) → insere texto em posição
.erase(pos, qtd) → apaga parte do texto
.replace(pos, qtd, str) → substitui parte
.substr(pos, qtd) → extrai substring
.find(str) → retorna posição da primeira ocorrência
.rfind(str) → busca da direita pra esquerda
.compare(str) → compara strings
.c_str() → converte para const char*

Vetores (std::vector)

.push_back(x) → adiciona elemento ao final
.pop_back() → remove o último elemento
.size() → número de elementos
.empty() → verifica se está vazio
.clear() → apaga todos os elementos
.at(i) → acessa elemento com verificação de limite
.front() → primeiro elemento
.back() → último elemento
.resize(n) → altera tamanho
.insert(pos, val) → insere em posição específica

Mapas e Conjuntos (STL)

std::map<K,V> → associa chaves a valores
.insert({k,v}) → adiciona par chave/valor
.erase(k) → remove elemento pela chave
.find(k) → retorna iterador para chave
.size() → número de elementos
std::set<T> → armazena valores únicos ordenados
.count(x) → verifica se elemento existe
.clear() → limpa o conjunto
